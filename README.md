## 🖥️ 스파르타슈팅클럽

+ [플레이 영상](https://youtu.be/cc5jiGRsd6Y)
+ 스파르타 3D FPS 연습장
+ 이동 [W, A, S, D] / 달리기 [LShift] / 걷기 [LCtrl] / 앉기 [C]
+ 발사 [LMB] / 조준 [RMB] / 재장전 [R]
+ 설정 [O] / 총기관리 [I]

## 📆 Develop Schedule

* 24.01.02 ~ 24.01.09

- `Unity 2022.3.2`
- **IDE** : Visual Studio 2022
- **VCS** : Git (GitHub Desktop)
- **Resolution** :	1920 x 1080 `FHD`
